### Para ejecutar C++ en Windows:

```
cd FILE_PATH
g++ .\NOMBRE_ARCHIVO.cpp -lpthread -o app

```

### Para ejecutar C en Windows:

```
cd FILE_PATH
gcc .\NOMBRE_ARCHIVO.c -lpthread -o app
.\app.exe
```
